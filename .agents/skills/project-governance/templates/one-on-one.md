# L3 一对一脚本（每人 20 min / 周）

> 路径：`submission/pm_private/11_<工号>_<YYYY-MM-DD>.md`
> 仅 PM 可访问，`.gitignore` 已拦截。
> 不替代 standup、不讨论 ticket 进度——专门捞 standup 捞不到的东西。

## 开场（3 min）

固定 3 句问候，不寒暄项目：
1. "上周/本周哪一刻最有成就感？"
2. "上周/本周哪一刻最想拍桌？"
3. "你睡眠/精力打分（1-10）这周变了吗？"

## 主轴（12 min · PM 从下表抽 3 题）

| 编号 | 问题 | 想捞的东西 |
|---|---|---|
| Q1 | 同事谁拖你后腿了？具体哪件事？ | 跨职能内耗、协作摩擦 |
| Q2 | 你看到谁的 PR/代码不太对劲，但没在公共圈说？ | 静默的质量风险 |
| Q3 | 如果本周只能保 1 个 P0，你保哪个？为什么？ | ta 的真实优先级判断 |
| Q4 | 你心里估的 W4 上线概率是多少？比上周升还是降？ | 个人风险体感 |
| Q5 | 哪个工具/资源缺位让你慢了 1 天以上？ | 资源缺口 |
| Q6 | 有没有你觉得 PM 该知道但没人说的事？ | 房间里的大象 |
| Q7 | 业务方/sponsor 这周对你有什么直接接触？ | 越级沟通信号 |
| Q8 | 你最近想学/想做的方向是什么？ | 长线动机、留人 |
| Q9 | 哪个工单写得让你最难下手？ | PM 自身工单质量回声 |
| Q10 | 你觉得下次发版前最容易踩的坑是什么？ | 隐性风险 |

## 收口（5 min）

```
[PM 重复] 你说的关键 3 点是: ...
[PM 承诺] 我下周做这 1 件事来帮你: ...
[ta 承诺] 你下周做这 1 件事让自己更快/更稳: ...
[下次见] 同时间、同地点
```

## PM 私密笔记格式

```markdown
# 1on1 with <工号> · 2026-MM-DD

## 公开可流转（征得本人同意后可入 RAID / decision log）
- 风险 1: ...
- 改进项 1: ...

## 保密（仅 PM 留底）
- 情绪/动机/团队人际: ...
- 健康/家庭/职业意向: ...
- PM 自我反思: ...

## Action（PM 单方承诺，下周必兑现）
- [ ] ...

## 下次抽题
- 上次抽: Q1, Q3, Q7
- 下次抽: Q2, Q5, Q10
```

## 红线

- **绝不在周会/standup 里复述 1on1 原话**，除非本人当场同意。
- **绝不把 1on1 笔记写在公共仓库**，必须落在 `pm_private/`。
- **PM 承诺的事 100% 兑现**，一次失约会让 1on1 系统永久失效。
