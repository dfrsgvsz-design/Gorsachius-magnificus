# L2 周会模板（每周五 16:00–16:45）

> 路径：`submission/reports/weekly/W<n>_meeting_minutes.md`
> 硬限时 45 min。超时即散，PM 罚自己改下周议程。

## 议程（固定 5 段）

| 时间 | 环节 | 主持 |
|---|---|---|
| 0–05 | 上周 action items 闭环检查 | PM |
| 05–20 | 每人 3 min：进度 + 里程碑 + 风险 | A→B→C→D→E |
| 20–30 | 跨职能依赖冲突仲裁 | PM |
| 30–40 | RAID Log 逐条刷新（新增 / 升级 / 关闭） | PM |
| 40–45 | 下周 P0 锁定 + 业务方需拍板事项 | PM |

## 个人 3 分钟脚本（每人照念）

```
[本周起扣] 周一承诺的 P0 是 <列表>
[本周底] 实际扣下 <X/Y>
[里程碑] 本周是否沉下预定里程碑 (Yes/No + 证据链接)
[发现的新风险] 上上周不知道但本周发现的 1-3 条
[需 PM 决断] 你听不明白 / 听明白了不能拍板的问题
[下周承诺] 1–3 个 P0（保守估计，不画大饼）
```

## 反内卷红线（PM 现场拦截）

- 任何"差不多" → 立刻问"差多少 %"
- 任何"应该可以" → 立刻问"靠什么验证"
- 任何"等 XX 给我" → 立刻问"等了多久 / 升级了没"
- 任何"下周补上" → 立刻问"为什么本周补不上"
