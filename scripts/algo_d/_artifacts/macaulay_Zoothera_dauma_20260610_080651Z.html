Macaulay Library top-up for Zoothera dauma

Open Macaulay catalog search

Cornell Lab Terms of Use: Macaulay Library media is downloadable for research/education only with attribution. Bulk programmatic download is not permitted without a data partner agreement. This script therefore never auto-downloads from Macaulay; download manually with your Cornell Lab account.

After downloading

  1. Save the audio files to one folder, e.g. F:\ml_dauma_dump\
  2. (Optional) fill in macaulay_Zoothera_dauma_20260610_080651Z.csv with asset_id / quality / country
  3. Run: 
    python scripts/algo_d/topup_external_sources.py import \
  --species "Zoothera dauma" \
  --folder F:\ml_dauma_dump\ \
  --source macaulay

Asset URL pattern (if you have asset IDs)

Direct download URL for a known asset id {asset_id}:

https://cdn.download.ams.birds.cornell.edu/api/v2/asset/{asset_id}/audio

Attribution requirement

Every imported Macaulay asset gets "license": "Macaulay/Cornell-Lab" and "attribution": "Macaulay Library, Cornell Lab of Ornithology, asset ML<ID>" in the manifest. Honor these when publishing trained-model outputs.
